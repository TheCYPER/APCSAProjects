# Project 1 - A calculator for quadratic equation
## Objective
* 完成一个计算器，计算**一元二次方程式**
* 对于一个形式为 0 = ax^2 + bx + c 的一元二次方程式
* 它的两个解为：
* x1 = (-b + (b^2 - 4a*c) ^ 1/2)/2a
* x2 = (-b - (b^2 - 4a*c) ^ 1/2)/2a
* 所以我们如果知道 **a, b, c**
* 我们就可以解这个方程了
* 那么我们如何编写出一个这个的Java程序呢？
